# QAXGitLab
奇安信项目GitLab自托管工具
,GitLab是整个软件开发生命周期中的单个应用程序。项目中使用这个工具实现从项目计划和源代码管理到CI / CD，监视和安全。
<h2>GitLab配置：</h2>
GitLab，RubyGems，PostgreSQL，Nginx以及从上游Omnibus软件包安装的所有其他必需组件。
在firstboot上设置GitLab管理员用户（'root'）密码和电子邮件。

<h2>apt更新</h2>
apt安装gitlab-ce

<h2>凭证（首次启动时设置的密码）</h2>
Webmin，SSH：用户名root
GitLab：用户名root
